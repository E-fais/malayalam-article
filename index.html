Introduction

Git, GitHub എന്നിവ version control , Collaborative Software Development എന്നീ കാര്യങ്ങൾക്ക്‌ ഉപയോഗിക്കുന്ന ഉപകരണങ്ങളാണ്. ഒരു തുടക്കക്കാരന് git പഠിക്കാൻ ഘട്ടം ഘട്ടമായുള്ള ഗൈഡ് ഇതാ:
Step 1: Git ഇൻസ്റ്റാൾ ചെയ്യുക നിങ്ങൾക്ക് ഔദ്യോഗിക Git വെബ്‌സൈറ്റിൽ നിന്ന് (https://git-scm.com/) Git ഡൗൺലോഡ് ചെയ്ത് ഇൻസ്റ്റാൾ ചെയ്യാം. നിങ്ങളുടെ ഓപ്പറേറ്റിംഗ് സിസ്റ്റത്തിനായുള്ള ഇൻസ്റ്റാളേഷൻ നിർദ്ദേശങ്ങൾ പാലിക്കുക.
Step 2: Git കോൺഫിഗർ ചെയ്യുക ഇൻസ്റ്റാളേഷന് ശേഷം, നിങ്ങളുടെ പേരും ഇമെയിൽ വിലാസവും ഉപയോഗിച്ച് Git കോൺഫിഗർ ചെയ്യേണ്ടതുണ്ട്. ഒരു ടെർമിനൽ അല്ലെങ്കിൽ കമാൻഡ് പ്രോംപ്റ്റ് തുറന്ന് ഇനിപ്പറയുന്ന കമാൻഡുകൾ Type ചെയ്യുക,
git config --global user.name "Your Name" git config --global user.email you@example.com
"your name", "you@example.com" എന്നിവ നിങ്ങളുടെ സ്വന്തം വിവരങ്ങൾ കൊടുക്കുക:. ഇനി git ഉപയോഗിച്ച്‌ തുടങ്ങാം.
